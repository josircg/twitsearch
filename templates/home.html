O que é ?

O Twitter Monitor é um sistema que permite que um pesquisador capture dados do Twitter a partir de critérios de busca

Como funciona?


1) O Pesquisador cria um Projeto e define os critérios de busca e uma data inicial e final
2) O sistema inicia a coleta e armazena os tweets coletados em um banco de dados relacional
3) A medida que a coleta é realizada, o pesquiador pode em tempo real analisar os resultados encontrados eincluir novos critérios de busca
4) Ao final do processo, o sistema emite relatórios de totalização e permite que você exporte os dados para softwares de análise

O sistema é gratuito para uso acadêmico
Se você tem já recebeu credenciais para utilizar o sistema, Clique aqui para acessá-lo
Se você é um pesquisador e tem interesse em utilizar a ferramenta, entre em contato
